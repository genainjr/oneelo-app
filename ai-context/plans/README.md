# Índice de Planos

Este diretório reúne planos de desenvolvimento, desenhos de solução, auditorias e relatórios de execução.

Os documentos abaixo estão ordenados pela data em que foram adicionados ao repositório, do mais recente para o mais antigo. A data representa o primeiro commit do arquivo no histórico atual do Git; documentos ainda não commitados usam a data de criação local.

Ao adicionar um novo documento, inclua-o no início deste índice com a data, o link e o título principal.
